# Scooter Rental Management System

## Description
This project is a simple scooter rental system. It allows users to view scooters, reserve them, rent them, and return them. The system also manages stations, scooter status, and maintenance.

## Features
- View available scooters
- Reserve scooters
- Rent and return scooters
- Calculate rental cost based on duration and scooter type
- Prevent multiple users renting the same scooter
- Handle reservation conflicts
- Check station capacity
- Manage scooter maintenance

## Technologies Used
- Python
- Object-Oriented Programming (OOP)

## How to Run
1. Open the project folder in VS Code
2. Make sure Python is installed
3. Run the file: python3 main.py



## File Structure
- main.py → runs the system
- classes.py → contains all classes
- diagrams/ → contains UML diagrams
- report.pdf → project report

## OOP Concepts Used
- Encapsulation: data and methods are inside classes
- Inheritance: Member class extends User class
- Abstraction: system is divided into simple classes
